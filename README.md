## 🛠️ How to Use

1. **Clone the repo:**
   ```bash
   git clone https://github.com/AbinAbey/Network-Lab-S6-CSE.git
   cd Network-Lab-S6-CSE


2. **Run the codes:**

   * For Python programs:

     ```bash
     python3 filename.py
     ```
   * For C programs:

     ```bash
     gcc filename.c -o output
     ./output
     ```
